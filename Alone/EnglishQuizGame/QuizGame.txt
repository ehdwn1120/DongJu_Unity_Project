QuizGame
퀴즈 로직은 외국 유튜브에 The Game Guy 채널에서 영어 자막으로 배웠다.

기본 형태 : 단순 클릭 퀴즈 게임

이번 프로젝트에서 배운 것

1. 스크립트를 나눠서 하는 것도 하나의 방법이다.
2. if 문을 잘 사용하면 정말 편리하다
3. 씬 이동시 오디오를 포함한 오브젝트가 사라진다.
	DontDestroyOnLoad 함수 사용


추가한 기능

1. 메인메뉴에서 게임 시작 버튼 클릭 시 게임 씬으로 넘어가기
2. 나가기 버튼을 누르면 게임을 off
3. 게임 화면에서 메인메뉴 씬으로 넘어가기
4. 클릭 사운드
5. BGM
6. 옵션 패널을 만든 후 사운드 클릭 시 on/off
7. 슬라이더로 볼륨 조절하기 - 문제점: 처음 클릭시 사운드가 커짐


후에 추가할 기능

기본으로 문제를 더 넣기
1. 배경음악 여러개 넣고 옵션에서 또는 메인화면에서 음악 바꾸기
2. 레벨을 만들고 정답이 몇개 이상일 경우 다음 레벨
